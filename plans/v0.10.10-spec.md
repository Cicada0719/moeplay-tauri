# v0.10.10 详细规格：主题系统扩展

## 1. 主题工具（`src/lib/utils/theme.ts`）

```ts
export type AppTheme = "dark" | "light" | "sakura" | "system" | "black" | "contrast";

export const APP_THEMES: { id: AppTheme; label: string; icon: string }[] = [
  { id: "dark", label: "深色", icon: "home" },
  { id: "light", label: "浅色", icon: "lightbulb" },
  { id: "sakura", label: "樱夜", icon: "heart" },
  { id: "system", label: "跟随系统", icon: "monitor" },
  { id: "black", label: "纯黑", icon: "moon" },
  { id: "contrast", label: "高对比", icon: "contrast" },
];

export function isValidTheme(value: string): value is AppTheme;
export function resolveTheme(theme: AppTheme): Exclude<AppTheme, "system">;
export function applyTheme(theme: AppTheme): void;
export function loadStoredTheme(): AppTheme;
```

### 1.1 `resolveTheme`

- `"system"` 通过 `window.matchMedia('(prefers-color-scheme: dark)').matches` 返回 `"dark"` 或 `"light"`。
- 其他主题原样返回。

### 1.2 `applyTheme`

- 计算生效主题 `effective = resolveTheme(theme)`。
- 设置 `document.documentElement.setAttribute("data-theme", effective)`。
- 将用户偏好 `theme` 写入 `localStorage.setItem("moegame-theme", theme)`。
- 当 `theme === "system"` 时，挂载 `matchMedia` 监听器，系统明暗变化时自动重新应用。
- 当切换到非 system 主题时，移除监听器，避免副作用泄漏。

### 1.3 `loadStoredTheme`

- 读取 `localStorage.getItem("moegame-theme")`。
- 校验合法性；非法或为空时返回 `"dark"`。

## 2. Settings Store 改造

- `defaultSettings.theme` 改为 `loadStoredTheme()`。
- `load()` 与 `save()` 中调用 `applyTheme(_settings.theme)` 替代原 `applyTheme` 内联函数。
- `toggleTheme()` 保持为 dark / light 切换（不改变现有快捷键行为）。

## 3. 设置页主题选择器

- 删除硬编码 `themes` 数组，改用 `APP_THEMES`。
- 按钮图标直接使用 `theme.icon`；无需特殊处理 system。
- 点击时仍调用 `settingsStore.save({ ...settingsStore.settings, theme })`。

## 4. 图标扩展

在 `Icon.svelte` 的 `icons` 对象中新增：

- `moon`: 月牙 SVG path。
- `contrast`: 半圆对比 SVG path。

## 5. CSS 新增主题

### 5.1 纯黑 `black`

以 dark 变量为基准，覆盖为更暗的 OLED 友好色：

- `--bg-void`: `#000000`
- `--bg-deep`: `#000000`
- `--bg`: `#050505`
- `--bg-surface`: `#050505`
- `--bg-secondary`: `#080808`
- `--bg-card`: `#0a0a0a`
- `--bg-hover`: `#101010`
- `--text-primary`: `#ffffff`
- `--text-secondary`: `#b0b0b0`
- `--text-muted`: `#555555`
- `--border`: `rgba(255,255,255,0.10)`
- `--border-hover`: `rgba(255,255,255,0.18)`
- `--glass-bg`: `rgba(5,5,5,0.82)`
- 调整 aura 面板与背景色为更暗版本。

### 5.2 高对比 `contrast`

强调可访问性：

- `--bg-void`: `#000000`
- `--bg`: `#000000`
- `--bg-card`: `#0a0a0a`
- `--text-primary`: `#ffffff`
- `--text-secondary`: `#e0e0e0`
- `--text-muted`: `#a0a0a0`
- `--border`: `rgba(255,255,255,0.25)`
- `--border-hover`: `rgba(255,255,255,0.45)`
- `--accent`: `#ff4d7a`
- `--accent-hi`: `#ff7a9b`
- `--focus-ring`: `0 0 0 3px #fff`
- 适当提高 aura 线条与面板边框对比度。

## 6. 测试规格

`src/lib/utils/theme.test.ts` 覆盖：

1. `isValidTheme`：识别 6 种合法主题，拒绝非法字符串。
2. `resolveTheme`：system 在暗色/亮色媒体查询下分别解析为 dark/light；非 system 主题原样返回。
3. `loadStoredTheme`：返回存储的合法主题；非法值回退 dark。
4. `applyTheme`：正确设置 `data-theme` 与 localStorage（通过 DOM/Storage mock）。
