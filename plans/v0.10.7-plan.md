# v0.10.7 优化计划：接入全局键盘快捷键系统

> 版本：0.10.6 → 0.10.7  
> 目标：让每个按键都能驱动界面，减少用户在手柄/鼠标与键盘之间的切换成本。

## 背景

当前应用只有 Escape 返回首页这一个全局快捷键，而大量高频操作（切换视图、聚焦搜索、打开工具）都依赖鼠标点击底部 Dock 或工具抽屉。对于大屏/全屏使用场景，键盘快捷键能显著提升操作效率。本轮引入 Svelte 原生快捷键库，建立可扩展、可发现的全局快捷键体系。

## 本轮整合的 GitHub 优秀项目

- [svelte-put/shortcut](https://github.com/svelte-put/shortcut) — 专为 Svelte 5 Runes 模式设计的快捷键 action，支持修饰键、单键、组合键、code-based 匹配。

## 验收标准

1. `npm run test:unit` 通过。
2. `npm run check` 无新增错误。
3. 全局可用以下快捷键：
   - `1`~`5`：切换 Dock 前 5 个主视图（游戏库、继续、番剧、漫画、工具抽屉）
   - `?`：打开/关闭快捷键帮助
   - `/` 或 `Ctrl/Cmd+K`：在游戏库页聚焦顶部搜索
   - `Escape`：保持返回首页行为
4. 帮助浮层列出所有可用快捷键及说明。
5. 版本号统一为 0.10.7。
