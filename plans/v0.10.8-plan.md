# v0.10.8 优化计划：统计页接入 Chart.js 图表

> 版本：0.10.7 → 0.10.8  
> 目标：把当前基于 SVG 手绘的统计图表替换为可交互、可响应、信息密度更高的 Chart.js 图表，让每一个统计数字都能讲述趋势。

## 背景

当前 StatsPage 的「月度热力图」和「完成率圆环」使用内联 SVG 与 CSS 渐变实现，维护成本高、交互弱、可访问性有限。本轮引入 Chart.js 这一成熟的开源图表库，将统计页升级为真正的数据可视化仪表盘：

- 月度趋势改为 Chart.js 折线图，支持 hover 提示、平滑曲线。
- 状态分布改为 Chart.js 水平条形图，直观对比各状态数量。
- 完成率改为 Chart.js doughnut 图，保留中心百分比。
- 热门标签/开发商保持现有列表，但增加空状态引导。

## 本轮整合的 GitHub 优秀项目

- [chartjs/Chart.js](https://github.com/chartjs/Chart.js) — 流行、稳定、框架无关的 JavaScript 图表库，通过 canvas 渲染，可轻松嵌入 Svelte 5 组件。

## 改动范围

| 类别 | 文件/模块 | 说明 |
|------|-----------|------|
| 新增依赖 | `package.json` | `chart.js` |
| 新增组件 | `src/lib/components/ui/Chart.svelte` | Chart.js 的 Svelte 5 封装，自动销毁、响应式重绘 |
| 新增工具 | `src/lib/utils/chart.ts` | 图表数据转换（月度趋势、状态分布、完成率） |
| 新增测试 | `src/lib/utils/chart.test.ts` | 数据转换函数测试 |
| 改造页面 | `src/lib/components/StatsPage.svelte` | 替换 SVG 图表为 Chart 组件 |
| 版本号 | 配置文件 + CHANGELOG | 0.10.7 → 0.10.8 |

## 验收标准

1. `npm run test:unit` 通过。
2. `npm run check` 无新增错误。
3. `npm run build` 成功。
4. StatsPage 的月度趋势、状态分布、完成率均使用 Chart.js 渲染。
5. 图表在暗黑主题下颜色协调，hover 提示可读。
6. 组件卸载时 Chart.js 实例被正确销毁，无内存泄漏。
7. 版本号统一为 0.10.8。
