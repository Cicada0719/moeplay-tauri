## External Release Blocker

The repository cannot manufacture or rotate the updater private key. Repository administrators must configure:

- `TAURI_SIGNING_PRIVATE_KEY`
- `TAURI_SIGNING_PRIVATE_KEY_PASSWORD` when the key is encrypted

The private key must match the public updater key in `src-tauri/tauri.conf.json`. Without it, the release workflow intentionally creates an installer-only draft labelled `UPDATER BLOCKED` and fails promotion.
