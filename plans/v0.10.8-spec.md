# v0.10.8 详细规格：统计页 Chart.js 图表

## 1. Chart.svelte 组件

```svelte
<Chart type="line" data={...} options={...} />
```

- Props:
  - `type: "line" | "bar" | "doughnut"`
  - `data: ChartData`
  - `options?: ChartOptions`
- 使用 `<canvas bind:this={canvas}>`。
- `onMount` 时创建 Chart.js 实例。
- `$effect` 监听 `data` / `options` 变化，调用 `chart.update("none")` 或 `chart.destroy()` + 重建。
- `onDestroy` 时 `chart.destroy()`。

## 2. 数据转换（src/lib/utils/chart.ts）

### 2.1 月度趋势

```ts
export function buildMonthlyTrendData(items: { month: string; hours: number }[])
```

- 取最近 12 个月。
- x 轴标签格式：`2026-07` → `7月`。
- y 轴：小时数。
- 数据集一条，颜色使用 CSS 变量 `--aura-data-a` / `--aura-data-b` 的近似值。

### 2.2 状态分布

```ts
export function buildStatusDistributionData(
  distribution: { status: string; count: number }[],
  statusLabels: Record<string, string>
)
```

- 过滤 count > 0 的状态。
- 按 count 降序排列。
- 生成水平条形图数据。

### 2.3 完成率

```ts
export function buildCompletionDoughnutData(rate: number)
```

- 两个数据段：完成率、剩余率。
- 颜色：强调色 + 边框色。

## 3. StatsPage 改造

### 3.1 移除

- `donutSvg` ref
- `trendPoints` 函数
- 内联 SVG donut / trend-line / heatmap

### 3.2 新增

- 引入 `Chart` 组件与 `buildMonthlyTrendData`、`buildStatusDistributionData`、`buildCompletionDoughnutData`。
- 完成率卡片：使用 doughnut Chart，中心叠加百分比文字。
- 状态分布卡片：使用水平 bar Chart。
- 月度趋势卡片：使用 line Chart。

### 3.3 主题适配

- Chart.js 默认颜色通过 options 显式设置为 CSS 变量或硬编码的暗色值。
- Tooltip 背景设为 `rgba(10,13,20,0.95)`，文字 `#fff`。
- Grid 颜色 `rgba(255,255,255,0.06)`。

## 4. 测试规格

`src/lib/utils/chart.test.ts` 覆盖：

1. `buildMonthlyTrendData`：返回正确数量的 labels/datasets，空输入处理。
2. `buildStatusDistributionData`：过滤零值、按 count 排序、标签本地化。
3. `buildCompletionDoughnutData`：rate 0/50/100 时的分段值。
