# Stage 4 / v0.13.0 — Unified Source Center

## Scope
- Unified Anime, Comic and external-runtime source descriptors with safe fields only: provider ID, media type, kind, capabilities, enabled/priority, health/latency/check time/auth/runtime state.
- SQLite schema v7 `source_preferences` (`provider_id`, `media_type`, `enabled`, `priority`, `updated_at`). Credentials remain in SecretStore.
- Backend commands: `list_source_descriptors`, `update_source_preference`, `verify_source`, `verify_sources_batch`, `reset_source_health`, `refresh_extension_index`, `get_extension_index_snapshot`.
- Metadata-only extension directory with endpoint validation, ETag/Last-Modified, 24h TTL and offline snapshot. Never download or execute extension/APK/plugin code.
- Shared selection policy excludes disabled/circuit-open entries, ranks health/latency/priority, and permits manual selection only within policy boundaries.
- Source Center UI for filter, enablement, priority, verification, health diagnostics and offline index state.

## Non-goals
- Suwayomi connectivity and extension execution (Stage 5).
- Third-party binary/plugin execution.
- Provider credential persistence outside SecretStore.

## Gates
- Command-contract update, schema migration tests, source policy and extension-index fixtures, Svelte/unit/Playwright coverage, full Rust/frontend/audit gates.
