# 萌游 MoeGame v0.19.9

## 窗口模式权限修复

- 0.19.8 的窗口模式修复未真正生效：窗口权限清单缺少 `set-fullscreen` 等写权限，前端全屏切换全部被系统权限层拒绝，窗口始终停留在全屏。本版本补齐权限，"窗口模式"现在真正可用。
- 启动后窗口为 1200×800 居中普通窗口，可自由拖拽边框调整大小。
- 顺带修复顶部导航手动全屏切换按钮失效的问题（同一权限缺失导致）。

## 发布与兼容性

- 版本号已统一更新至 0.19.9。
- 本版本面向 Windows x64，提供 **NSIS 安装版**（推荐，支持应用内签名自动更新）、**MSI 安装版**（适合集中部署）和 **便携包**，所有安装包均已签名。

**不包含 APK 或 AAB。**
