# 萌游 MoeGame v0.19.6

## 海报式封面电影墙

- “继续游玩”改为海报式封面电影墙：当前游戏为 2×2 大海报块，其余封面按列对向两侧多块铺开。
- 滚轮、拖拽、方向键整墙滑动切换；封面裁切拼接，活动海报高亮放大。

## 底部按钮修复

- 修复“继续游玩”页底部“继续游戏 / 打开档案”按钮点击不生效的问题：拖拽指针捕获不再拦截按钮点击。

## 记录页同步小说漫画

- 记录页（个人媒体年鉴）活动流接入小说阅读历史，与游戏、番剧、漫画统一排序展示。
- 新增“小说历史”统计卡与媒体分布计数；点击小说记录可直接回到对应章节续读。

## 发布与兼容性

- 版本号已统一更新至 0.19.6。
- 本版本面向 Windows x64，提供 **NSIS 安装版**（推荐，支持应用内签名自动更新）、**MSI 安装版**（适合集中部署）和 **便携包**，所有安装包均已签名。

**不包含 APK 或 AAB。**
