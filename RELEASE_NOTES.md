# 萌游 MoeGame v0.19.4

## Galgame 启动路径修复

- 用户在导入或编辑档案中选择的汉化版 EXE 现在拥有最高优先级。
- 引擎探测只负责识别引擎与区域启动方式，不再重新挑选同目录中的其他 EXE。
- 删除启动后自动回写猜测路径的行为，避免汉化路径再次被改回原版。
- 本地启动路径优先于历史遗留的 Steam、Epic、Ubisoft 等平台 URI；平台游戏本身仍保持协议启动兼容。
- 编辑路径后会同步新的安装目录，引擎探测也跟随所选 EXE 的父目录。
- 空路径、失效路径和启动失败会给出明确反馈，不再显示虚假的启动成功提示。

## 游戏库与专注模式

- 专注模式开关进一步缩小、上移，并避让底部导航与手柄提示栏。
- 移动端、低高度窗口、安全区域和减少动画模式均提供对应布局。
- 超长中文、日文、英文及无空格标题会正确截断或断行。
- 卡片操作区不再随标题压缩，收藏、删除等按钮始终保留完整空间。
- 详情页标题最多显示两行，并在低高度窗口收紧标签与按钮间距。

## 发布与兼容性

- 版本号已统一更新至 0.19.4。
- 修复 GitHub Release 对中文 NSIS 安装包文件名的匹配逻辑。
- 本版本面向 Windows x64，提供 NSIS、MSI、便携包以及签名自动更新资源。

**不包含 APK 或 AAB。**
