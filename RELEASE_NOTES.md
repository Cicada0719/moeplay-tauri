# 萌游 MoeGame v0.19.5

## 立方舞台长标题修复

- 修复超长游戏标题（罗马音长名、中日文长名）把“继续游戏 / 打开档案 / 收藏”按钮和游戏目录挤出档案面板的问题。
- 标题按长度分三档自动缩小字号，并限制最大行数，操作区始终可见、可点击。

## 小说搜索提速

- “全部书源”搜索改为逐源并发、先到先展示，慢源不再拖垮整体，最快的结果立刻可见。
- 单源搜索超时从 30–35 秒收紧到 10 秒；HTTP 客户端全局共享并复用连接。
- 搜索中可随时取消，并实时显示已返回书源进度。

## 继续游玩封面滚筒

- 游戏库“影像序列”升级为“继续游玩”模块：多张封面裁切后拼接在水平滚筒上，以俯视角 3D 滚动。
- 支持滚轮、拖拽与方向键转动；滚筒装入带上下出纸口的盒体中。
- 页面顶部旧的继续游玩横栏暂时移除，模式切换器第三个标签同步更名。

## 发布与兼容性

- 版本号已统一更新至 0.19.5。
- 本版本面向 Windows x64，提供 NSIS、便携包以及签名自动更新资源。

**不包含 APK 或 AAB。**
