# 萌游 MoeGame v0.19.7

## 摇杆直切游戏

- 主界面（立方舞台 / 继续游玩）推摇杆或十字键 ↑↓ 即可直接切换选中游戏，视觉模式 ←→ 切换当前游戏媒体，与键盘操作完全一致。
- 不再需要逐格移动焦点到目录按钮再按 A——拿起手柄推一下就能浏览整个游戏库。

## 手柄导航更稳

- 手柄焦点锁定在舞台根元素：连续切换游戏不再因按钮重渲染而跳回顶部导航。
- 按 B 退出舞台回到顶部导航；LB/RB 切换主页面后直接落在舞台，可立即继续推摇杆。
- A 打开当前游戏档案，Y 收藏当前游戏，手柄提示栏在舞台焦点下显示正确操作。
- 修复合成键盘事件误判导致的输入模式回跳；几何导航滚动改为即时响应，连发不再卡顿。

## 发布与兼容性

- 版本号已统一更新至 0.19.7。
- 本版本面向 Windows x64，提供 **NSIS 安装版**（推荐，支持应用内签名自动更新）、**MSI 安装版**（适合集中部署）和 **便携包**，所有安装包均已签名。

**不包含 APK 或 AAB。**
