# 萌游 MoeGame v0.19.8

## 窗口模式修复

- 修复"窗口模式"不生效：启动时窗口不再卡在原生全屏，窗口模式的应用现在带验证和多次重试，覆盖 Windows 窗口初始化早期全屏切换被静默忽略的情况。
- 修复切换"窗口模式"后窗口被最大化、无法拖拽调整大小的问题：现在立即还原为 1200×800 居中窗口，可自由拖拽边缘调整大小。
- 设置页即时生效与启动时生效共用同一套窗口模式逻辑，行为一致。

## 发布与兼容性

- 版本号已统一更新至 0.19.8。
- 本版本面向 Windows x64，提供 **NSIS 安装版**（推荐，支持应用内签名自动更新）、**MSI 安装版**（适合集中部署）和 **便携包**，所有安装包均已签名。

**不包含 APK 或 AAB。**
